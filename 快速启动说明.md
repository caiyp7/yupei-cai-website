# 🚀 快速启动说明

## 最简单的方式

**双击运行：`启动网站.bat`**

这个脚本会自动：
- ✅ 检查是否安装了 Node.js
- ✅ 安装所需的依赖包
- ✅ 启动开发服务器
- ✅ 在浏览器中打开网站

---

## 如果还没安装 Node.js

### 第一步：安装 Node.js

1. **访问 Node.js 官网**
   - 网址：https://nodejs.org/
   - 点击绿色的 "LTS" 版本下载（推荐 v18 或 v20）

2. **安装 Node.js**
   - 运行下载的安装程序
   - 一直点击"下一步"即可
   - 安装完成后**重启电脑**（或重启命令行窗口）

3. **验证安装**
   - 打开命令行窗口（PowerShell 或 CMD）
   - 输入：`node --version`
   - 如果显示版本号（如 v20.10.0），说明安装成功

---

## 手动启动步骤（如果已安装 Node.js）

### 步骤 1：打开命令行

- **Windows 10/11**: 按 `Win + R`，输入 `cmd` 或 `powershell`，回车

### 步骤 2：进入项目目录

在命令行中输入：

```bash
cd D:\download\yupei-cai-website
```

### 步骤 3：安装依赖（首次运行）

```bash
npm install
```

> ⏰ 这可能需要几分钟时间，请耐心等待

### 步骤 4：启动网站

```bash
npm run dev
```

### 步骤 5：打开浏览器

看到类似这样的输出后：

```
  VITE v5.x.x  ready in xxx ms
  
  ➜  Local:   http://localhost:5173/
  ➜  Network: use --host to expose
```

在浏览器中访问：**http://localhost:5173**

## 常见问题

### ❓ 提示 'npm' 或 'node' 不是内部或外部命令

**解决**：需要先安装 Node.js（见上面的"第一步"）

### ❓ 安装依赖很慢

**解决**：可以使用国内镜像源加速

```bash
npm config set registry https://registry.npmmirror.com
```

然后再运行 `npm install`

### ❓ 端口 5173 已被占用

**解决**：
1. 关闭占用端口的其他程序
2. 或者在命令行按 `Ctrl + C` 停止服务器后，修改配置使用其他端口

### ❓ 想停止服务器

**解决**：在命令行窗口按 `Ctrl + C`

### ❓ 想重新启动

**解决**：直接再次运行 `npm run dev` 即可
