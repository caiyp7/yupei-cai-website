# 个人网站项目说明

   - 更新论文的 PDF、arXiv、DOI 链接
   - 更新论文状态（Under Review / Accepted / Published）

5. **项目链接** (`src/pages/Projects.jsx`)

   - 添加项目演示视频链接

## 🎨 样式自定义

所有样式都在 `src/styles/` 目录下，每个页面有独立的 CSS 文件。

全局样式变量在 `src/styles/index.css` 中定义：

```css
:root {
  --primary-color: #2563eb;      /* 主色调 */
  --primary-dark: #1e40af;       /* 深色主调 */
  --secondary-color: #64748b;    /* 次要颜色 */
  --text-color: #1e293b;         /* 文字颜色 */
  --bg-color: #ffffff;           /* 背景色 */
  /* ... */
}
```

修改这些变量可以快速更改整个网站的颜色主题。

## ❓ 常见问题

**如何添加新的页面？**
1. 在 `src/pages/` 创建新组件
2. 在 `src/App.jsx` 添加路由
3. 在 `src/components/Navbar.jsx` 添加导航链接

**如何修改网站标题？**
编辑 `index.html` 中的 `<title>` 标签

**如何添加Google Analytics？**
在 `index.html` 的 `<head>` 部分添加 GA 代码
